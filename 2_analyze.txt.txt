Главная страница сайта обладает следующей структурой:
1) Шапка сайта id = "mw-page-base", основное меню меню сайта
2) Далее идёт блок основного контента id = "content", в который входит 
	2.1) Блок меню слева
	2.2) Блок со статьями id = "mw-content-text"
	2.3) В этом блоке отдельно можно выделить ссылки на доп.материалы div class = "main-wikimedia main-plainlist"
3) Футер, он же подвал id = "footer", с условиями использования сайта, описанием организации и информацией по сайту
